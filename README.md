# Dawnbot Discordbot

## Atention

You must use a .env file including a *discord bot token*, a *random message* and the bot admin *info*. (env files are to be used mostly with heroku (In this project), you may need to work arround if you want to use env.

## Info needed in the env file

MESSAGE: a random message
BOT_ADMIN_ID: the discord id from the bot admin
BOT_ADMIN_NAME: the name from the bot admin
TOKEN: the bot discord token
